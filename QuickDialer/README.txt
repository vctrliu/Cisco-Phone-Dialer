**You must have the XML file in the same directory as the executable for the program to work.

**FEATURES
Autoformat: Removes extraneous symbols/spaces, appends the "8" or "81" for outside lines automatically.
Commandline arguments: "Quickdialer.exe 7700" would call "7700". Use this feature to plug it into launchy.


**HOTKEYS:
ENTER or ALT+C: Start call
ESC: Clear input box
Alt+E: End call

**FUTURE ENHANCEMENTS (aka Current limitations):
-Incorporate a PAUSE character, this will allow you to dial a conference line, pause, and then enter the ID number automatically.
-Taskbar/context menu integration
